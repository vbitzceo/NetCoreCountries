# NodeJSUsingEventEmitter
IMPORTANT: This project is a learning tool. It's not the be all end all but it is a good guide on how to structure your .NET Core/5.0 Projects

## Motivation
A minimal app to allow others to explore the typical structure used in larger .NET applications. Here you will find Dependency Injection, Repository Pattern, Seperation of Concerns, 
and much more.

## Installation or Getting Started
 
  Setup SQLite:
  This app requires that you have installed SQLite. Find it here: https://sqlite.org/index.html

  Setup Visual Studio 2019:
  This app requires that you have installed Visual Studio 2019. Find it here: https://visualstudio.microsoft.com/vs/

  Clone the repository:
  git clone https://github.com/vbitzceo/NetCoreCountries.git
  
  Open in VS 2019
  Goto Tools > NuGet Package Manager > Package Manage Console 
  Type "update-database" and press return in the Package Manage Console (this will deploy the db)
  Click the Play button in Visual Studio
